# Airpwn
Airpwn is an automated wireless attack menu made for Debian. It has lots of capabilites and features which make attacking as easy as it can be.

### *Installation:*
```
git clone https://github.com/StateraDuo/airpwn
cd airpwn
./installAirpwn
```
